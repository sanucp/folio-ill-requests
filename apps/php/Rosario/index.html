APP de completado de estadisticas de Curriculum Builder - Universidad del Rosario




Instrucciones del Aplicativo

  • 1. Descarge los archivos de estadísticas de Curriculum Builder (dat.csv y studentdata.csv) desde el administrador de la herramienta (aquí), (Recuerde que estos archivos deben tener la extensión .CSV).
  • 2. Subir los archivos en el espacio de este aplicativo definido para cada uno de los archivos (son los que se encuentran a la izquierda de estas instrucciones); solo cargue uno a la vez y espere que el proceso termine para luego realizar el proceso con el otro.
  • 3. El aplicativo comenzara a realizar el proceso de suma de datos, esto puede demorar un tiempo dependiendo del tamaño del archivo.
  • 4. El aplicativo descargara automaticamente el archivo una vez terminado el procesamiento de datos, este quedara en la carpeta de descarga del navegador.
